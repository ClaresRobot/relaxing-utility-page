        Sobre esta página

        ¡Bienvenido/a a mi página web! Esta web sirve como "Utility page", y su proposito es tener diferentes tipos de utilidades que los usuarios puedan disfrutar. Actualmente incluye una lista de tareas, un pronóstico del clima, un bloc de notas, un reproductor de música, una calculadora, una cita del día y un lector de noticias. La idea es que sea un espacio versátil donde puedas acceder a varias herramientas útiles en un solo lugar, con un aspecto relajado y visualmente atractivo.

        Creación de la página

        Esta página fue creada enteramente en HTML, CSS y JavaScript. Se utilizó un framework de CSS para obtener el estilo característico de Windows 7, y varias APIs públicas para obtener diferentes datos. Es una demostración de mis habilidades en desarrollo Fullstack, un pequeño proyecto personal para prácticar desarrollo web general. Me plantearé añadir nuevas utilidades en el futuro, además de una traducción al inglés.
